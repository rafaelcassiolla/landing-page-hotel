# 🏨 Maison Soleil - Frontend Mentor Challenge

Este projeto é a minha resolução para um dos desafios propostos na plataforma [Frontend Mentor](https://www.frontendmentor.io/). O objetivo foi construir um dashboard/painel de confirmação de reserva de hotel moderno e responsivo, buscando a máxima fidelidade ao design original proposto pelo desafio.

---

## 🚀 Tecnologias Utilizadas

O projeto foi construído do zero focando em pixel-perfect design e boas práticas visuais:

* **HTML5:** Estruturação semântica e acessível de toda a página (sidebar, seções de conteúdo e cards informativos).
* **CSS3:** Estilização baseada no guia de estilo do desafio (cores, fontes, sombras e efeitos de sobreposição), uso de Flexbox/Grid para o layout e responsividade completa.

---

## 🛠️ O que foi praticado neste desafio?

* **Fidelidade ao Design:** Traduzir um layout complexo com elementos sobrepostos (como o recibo e o cartão de boas-vindas) em código real estruturado.
* **Componentização de Interface:** Criação de uma sidebar de navegação lateral fixa, exibição estilizada de recibo de estadia (`Receipt`), cards informativos de utilidades (Arrival, Wi-Fi, Breakfast) e um widget de clima local.
* **Design Responsivo:** Adaptação da interface para garantir uma ótima experiência tanto em desktops quanto em dispositivos móveis.
